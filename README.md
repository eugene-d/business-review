# business-review

##Vagrant deployment

- Add to hosts file "192.168.66.66  br.dev"
- Install virtualBox and vagrant [downloads](https://www.vagrantup.com/downloads.html)
```sh 
    $ vagrant box add hashicorp/precise64
```
```sh 
    $ vagrant up
```
- Then in browser [http://br.dev](http://br.dev)
